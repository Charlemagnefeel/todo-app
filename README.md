# 自用待办

一个离线安卓待办 App。数据保存在本机，APK 不申请联网权限。

## 功能

- 自定义任务类别，默认有室内、室外、定期。
- 短期任务排在长期任务前面。
- 普通任务完成后归档，可编辑、还原、删除。
- App 内支持同类别、同长短期任务长按拖动排序。
- 定期任务记录打卡次数、日期和距上次打卡时间，支持撤销最近一次打卡。
- 支持数据导出和导入，方便备份或换手机。
- 桌面小组件按类别显示，可选择哪些类别出现在小组件里。
- 小组件支持字号调节、上下滚动，短期和长期任务之间有对齐分隔线。
- 小组件内可直接完成普通任务，也可给定期任务打卡或撤销最近一次打卡。

## 构建

只下载 APK 安装的话，不需要 Android SDK 或 JDK。

下面这些命令是给想从源代码重新打包 APK 的人用的。开发环境需要 Android SDK 和 JDK 17+。

```bat
build-debug.bat
```

安装到已授权 USB 调试的手机：

```bat
install-debug.bat
```

VSCode 可以直接运行同名 Android 任务。
