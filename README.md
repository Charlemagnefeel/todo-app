# 自用待办

一个离线安卓待办 App。数据保存在本机，APK 不申请联网权限。

## 功能

- 自定义任务类别，默认有室内、室外、定期。
- 短期任务排在长期任务前面。
- 普通任务完成后归档，可编辑、还原、删除。
- 定期任务按打卡记录次数、日期和距上次打卡时间。
- 桌面小组件按类别显示，可选择哪些类别出现在小组件里。

## 构建

需要 Android SDK 和 JDK 17+。

```bat
build-debug.bat
```

安装到已授权 USB 调试的手机：

```bat
install-debug.bat
```

VSCode 可以直接运行同名 Android 任务。
